# BlogMan 
BlogMan是一个基于VSCode的博客管理插件，可以帮助你在VSCode中管理你的博客，可以按照时间, tag, 年份进行分类管理。
## 使用方法
使用Vscode打开存储博客md文件的文件夹，再点击BlogMan的图标，即可看到博客的分类信息。点击右上角，可以刷新内容，或者切换文章排序方式。
## 构建方案
```bash
npm install
npm run compile
vsce package
```
